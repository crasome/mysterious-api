Offers API

Get a list of offers

GET /offers?uid=:uid&pub0=:pub0&page=:page

Parameters

Name Description
uid User unique id
pub0 Custom parameters
page Page to request (pagination)

Response Fields

Name Description
offers A collection of offers
offers[id] Offer id
offers[title] Offer title
offers[thumbnail] Thumbnail object
offers[payout] Amount of virtual currency

Request

Headers

Api-Version: v1
Accept: application/vnd.api+json
Content-Type: application/vnd.api+json
Host: example.org
Cookie:

Route

GET /offers?uid=player1&pub0=campaign2&page=1

Query Parameters

uid: player1
pub0: campaign2
page: 1

curl "http://localhost:3000/offers?uid=player1&pub0=campaign2&page=1" -X GET \
	-H "Api-Version: v1" \
	-H "Accept: application/vnd.api+json" \
	-H "Content-Type: application/vnd.api+json" \
	-H "Host: example.org" \
	-H "Cookie: "
